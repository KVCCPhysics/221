Error Propagation - Finding the expected range

Even for kinematics (a simple case), propagating error leads to complications. If you attempt to go headlong into the math, you arrive at the following: (This includes clever trick which you will need for the first exam)

solving for range in 2D free fall without knowing time:

  • write out the kinematic equations \[x_f = x_i + v_i cos(\theta)\,t\] \[y_f = y_i + v_i sin(\theta)\,t + \frac12 a \, t^2\]
  • solve for time with the x equation \[t=\frac{x_f - x_i}{v_{i}cos(\theta)}\]
  • substitute this result into \(y_f\) equation \[y_f = y_i + v_i sin(\theta)\left(\frac{x_f - x_i}{v_{i}cos(\theta)}\right) + \frac12 a \, \left(\frac{x_f - x_i}{v_{i}cos(\theta)}\right)^2\] \[y_f = y_i (x_f - x_i) tan(\theta) + \frac12 a \, \left(\frac{x_f - x_i}{v_{i}cos(\theta)}\right)^2\]

this is a useful result to use for solving physics problems. But realize the part we are after for this experiment is actually the \((x_f - x_i\)) term, which is quadratic. Yikes! This is a show stopper and other methods must be used (sampling or other simulations).

The simplest sampling method is normally a Monte Carlo Simulation. This seems like a nice homework problem :)
